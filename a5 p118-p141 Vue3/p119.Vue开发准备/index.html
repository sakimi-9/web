Vue开发准备(教学视频)

 Vue官网.中文

安装vue cli工具 这个在导航栏生态系统里的工具链中vue cli用命令行安装

npm install -g @vue/cli 加c更快，有 -g 代表全局安装，安装完记得重启vscode
vue --version 验证vue是否安装成功
vue create my-project my-project是我的项目名字，可以改成别的名字，名字不能大写，用 - 来连接不同的单词
创建后怎么选择 用上下键来选择，空格来选中和取消选中，再回车确定
  1. 第三个是手动选择特性，选择第三个
  2. 回车后,选择1和3就行，把默认的倒数第三个(*) Linter / Formatter用空格取消，这个是代码规范化（多一个空格都会报错），选了这个很可能会有不必要的报错
  3. 回车后，然后选择vue 3.0版本
  4. 再回车后，选择默认的文件保存位置，不要改选项直接回车
  5. 最后一个，是问是否保存该预设，下次用项目名称来调用预设，选择n，暂时不用，之后会改,输入n,回车

cd vue-demo 进入vue-demo项目
npm run serve 运行该项目 不要加c
运行成功后 ctrl+点击 两个链接都行，进入项目网站，项目启动成功

另外 下载 vue代码高亮插件 Vue - Official 这样打开vue-demo里src中的 .vue 文件，代码启动高亮了
src是写源码的地方，其他的都是配置文件，暂时不要去动
