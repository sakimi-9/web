element-plus引入

element-plus官网
element-plus 描述 element-plus是饿了么基于vue向设计师和开发者提供的开源组件库，里面有各种组件的ui等等
第三方 命令行 安装 element-plus组件库属于第三方，需要命令行安装到项目根目录下
命令:cnpm install element-plus --save
完整引入 如果对打包后的文件大小不是很在乎，那么使用完整导入会更方便。 内存大，步骤少

在总入口main.js里，进行引入与挂载
import ElementPlus from 'element-plus' import 'element-plus/dist/index.css' app.use(ElementPlus)
按需引入 对项目内存大小敏感则用按需引入，但需要安装额外的插件。 内存小，步骤多

首先需要安装unplugin-vue-components 和 unplugin-auto-import这两款插件
命令:cnpm install -D unplugin-vue-components unplugin-auto-import 在vue.config.js中 将里面的代码换成 //官网的是vite ts形式的暂时不知道如何处理来适配，现在只能用完整 引入
使用方式 在element-plus官网，找到想要的组件，点击查看源代码，对包括组件含class的div容器复制，
因为已经对element-plus完整引入了，所以下面的js和css不用复制，只有有的组件的属性值需要用js定义一下
