字体图标

字体图标的本质是字体,(与图片相比)有以下优点

  1. 轻量性:加载速度块,减少http请求
  2. 灵活性:可以用CSS的font-()来调整样式
  3. 兼容性:网页字体适用所有现代浏览器,包括低版本IE浏览器
    字体图标,如何使用(以阿里巴巴矢量图标库为例 iconfont.com)
  1. 在矢量图标库下载相应图标到本地
    比如:在阿里巴巴的矢量图标库,将所需图标加入购物车,再全部添加进项目,点击上方的下载至本地(解压在合适的文件夹内)
  2. 调用外部样式
    用link调用文件夹里的 iconfont.css
  3. 引用类名
    打开文件夹里的html 选择想要图标 复制名称(去掉.号) 用这种格式引用
    class="iconfont(空格)名称"
  4. 样式设置
    对要修改样式的字体图标,通过以下形式修改
    .名称{声明}
