    打开服务端

    2. 新开一个命令行
  2. mongo
    打开客户端,连接服务端,然后就可以在 客户端 输入命令 进行与服务端交互

笔记 字段类型 那里 不要相信用户的输入 意思是 用户可能会输入非法数据,来破坏网站 所以要做好字段验证,检查好要录入的数据
笔记 条件控制 的 正则匹配 那里 有两种写法
  • / /
    简洁写法 但 解析能力有限 常用于 非变量 的正则匹配
  • new RegExp( )
    常规写法 常用于 变量 的正则匹配

正则匹配还有可选参数
比如: model.find({name: /a/i 或者 new RegExp('a','i') }) 表示 模糊查询 name中含有a(不区分大小写)的文档
mongoose 的代码模块化 通常是把 代码分成 3个部分
  • db文件夹
    存放 db.js(连接到数据库框架) 还有 confing.js(url配置)
  • models文件夹
    存放 XXmodel.js(各集合封装对象)
  • curd文件夹
    存放 app.js(增删改查的主文件-运行文件)
