node.js 是一款可以运行js的编译环境，一款软件/程序

node.js 作用

  1. 开发服务器应用 如 运行在服务器端的node.js ,对用户请求进行处理，比如请求url返回相应的资源
  2. 开发工具类应用 如 Babel vite webpack
  3. 开发桌面端应用
