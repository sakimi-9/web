path模块 各API

使用path模块也要跟使用fs模块一样，先用require导入
let path = require('path');

resolve() 方法
最常用		 两个参数(绝对路径，相对路径)
用于规范拼接绝对路径和相对路径 一般为 path.resolve(__dirname,'/文件全名')

__dirname+'/文件全名' 也可以拼接为 文件的绝对路径 但是不规范
可以理解为 resolve方法是将 __dirname+'/文件全名' 的拼接 规范化
sep 属性 返回当前操作系统的路径分割符 Windows \ mac /

path.sep
parse() 方法 参数传入路径

__filename 保存着 当前文件的绝对路径 但是不能拿来直接用 需要转义
先用__filename 获取路径 再用parse方法

let str_path = __filename


解析路径 返回 存储路径全部信息的object
path.parse(str_path)
以下方法都是返回 路径object里的数据 参数传入路径,解析路径返回存储路径信息object里的
  1. basename() 方法 文件全名
    path.basename(str_path)
  2. dirname() 方法 文件所在目录的绝对路径
    path.dirname(str_path)
  3. extname() 方法 文件扩展名
    path.extname(str_path)
