概念

HTTP协议 介绍 HTTP（hypertext transport protocol）协议；
中文叫超文本传输协议


是一种基于TCP/IP的应用层通信协议

这个协议详细规定了 浏览器 与 服务器 之间互相通信的规则。

协议中主要规定了两个方面的内容

客户端：用来向服务器发送数据，可以被称之为请求报文
服务端：向客户端返回数据，可以被称之为响应报文
报文：可以简单理解为就是一堆字符串

请求报文 组成

组成
  1. 请求行
  2. 请求头
  3. 空行
  4. 请求体
请求行 组成
  1. 请求方法
    • GET 常用 主要用于获取数据
    • PSOT 常用 主要用于新增数据
    • PUT/PATCH 主要用于更新数据
    • DELETE 主要用于删除数据
  2. 请求URL (统一资源定位器) 【定位资源在服务器的哪个位置】
      例如：http://www.baidu.com:80/index.html?a=100&b=200#logo
    • http： 协议（https、ftp、ssh等）
    • www.baidu.com 主机名(主机名=服务器名+域名) www是服务器名，万维网提供的 baidu.com是域名
      这个位置也可以是 主机名/域名/ip
    • 80 端口号
    • /index.html 路径 资源所在位置的具体路径
    • a=100&b=200 查询字符串 也就是额外上传参数
    • #logo 哈希（锚点链接）
  3. HTTP协议版本号
请求头 格式为键值对『头名：头值』

空行 就是空行，分隔请求体和上面的部分
请求体 请求体内容的格式是非常灵活的，

（可以是空）==> GET请求， 也就是请求报文为GET请求方法时 请求体是空的
（也可以是字符串，还可以是JSON）===> POST请求

字符串：keywords=手机&price=2000
JSON：{"keywords":"手机","price":2000}
