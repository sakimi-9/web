对象解构赋值

作用 对象中某个属性使用率非常高时，可以用对象解构赋值，把它单独拎出来，使用该属性时就等于使用对象中的该属性
格式
声明类型（var let const 等）		 声明类型 {属性集合} = 所属对象
单独拎出来的这个属性是所属这个对象

声明类型 {属性集合} = {属性:值，属性:值等}
单独拎出来的这个属性对应着对象里的这个键值对

({属性集合} = {属性:值})
这个不用写声明类型，避免了let 与 const的重复声明报错，有很大弊端
注意事项
  1. {}里的变量名必须是所属对象里有的（名称一样，且存在）
  2. 无声明类型的那种格式，很难读，而且容易出错，不如直接用 对象.属性名 这种原来的格式
  3. 声明对象时的声明类型用var ，对象解构赋值时的声明类型用另外两个，这样可以避免重复声明报错
