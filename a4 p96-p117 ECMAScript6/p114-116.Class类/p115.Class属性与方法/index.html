Class属性与方法

实例属性 类里的constructor构造函数和里面的属性，可以被类实例调用，被称为实例属性
实例属性里构造函数和实例方法的this指向，因为实例的属性值传参后赋值给this.属性，所以this.属性 this.方法的this都是指向实例
实例方法 类里的方法，可以被类实例调用，被称为实例方法
里面用this指向实例的属性值
静态方法 要创建静态方法，则要在创建的方法前 加上static关键字，静态方法在类里，静态方法是类的专属方法，实例不可调用不可继承（static 公共的，也就是不被实例特殊专用的）
调用静态方法:类名.方法

创建格式:
static 方法（）{任务代码块}

静态方法里this的指向，this永远指向被调用者，调用静态方法的是类，所以指向类，也就是类的专属，不是实例属性实例方法

另外 静态方法和实例方法的方法名可以相同，里面的this指向不同
静态属性 要给类创建静态属性，则要在类的外边(里面声明不了，没有构造函数承载)，进行声明，静态属性这是类专属的属性，实例不可调用不可继承
调用静态属性:类名.属性

创建格式:
类名.属性 = 值
