对象的扩展

属性简洁表达
  • 当属性名和属性值名称一样时（这是在调用外部的同名变量），可以省略属性值
    举例:var name="a" ; var object ={ name:name,(可省略成下面这个) name, }
  • 方法可以省略成方法名的形式出现
    举例:var object = { getname:function(){函数体}，（可简洁成下面这种） getname（）{函数体}， }
属性名表达式 属性名可以写在中括号里[]，[]可以识别语法，那么在[]里可以 调用外部变量名的值 写表达式
举例:var a=1,b=2;var name="zs" var object ={ [name]:38, [a+b]:"3" }
对象的扩展运算符... 可以将对象里的键值对全部拎出来（不能直接返回输出），拎 给另外的对象
格式:var object = {键值对}; var object1 = {键值对，键值对，...object}
