CSS 操作

三种用js通过Dom控制CSS的操作
元素节点的setAttribute()方法 格式:
标签变量名.setAttribute("style","声明(集合)")		 优点:最简单
将style属性与值声明集合 塞入 标签
元素节点的style属性 格式:
标签变量名.style.样式属性 = "样式属性的值"
或
标签style属性的变量名.样式属性="样式属性的值"		 优点:最清晰
将标签style属性里的各个样式属性赋值
元素节点里style的cssText属性 格式:
标签变量名.style.cssText = "声明(集合)"
或
标签style属性的变量名.cssText="声明(集合)"		 格式；7分简单3分清晰
将声明集合赋值给标签style属性里的cssText属性
注意赋值时等号左边的属性是小驼峰命名法
