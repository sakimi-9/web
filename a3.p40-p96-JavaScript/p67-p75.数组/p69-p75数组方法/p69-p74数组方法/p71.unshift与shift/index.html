数组方法 unshift与shift

unshift与shift,主要用于对数组头部元素的添加与删除，都会改变原数组

与push-pop 的作用相同但反向 push对标unshift , pop对标shift
添加 返回长度 ， 删除 返回该删除元素
unshift 作用:向原数组头部添加一个或多个元素，并返回新数组的长度

格式:arr.unshift(添加的元素)

unshift()括号里不写添加的元素，此时直接返回数组长度
arr.length === arr.unshift()
shift 作用:把原数组头部的一个元素删除，并返回删除的元素

格式:arr.shift()

注意，这个 写出来一次就会删除一次原数组，在while（）的括号里也一样，而unshift（）括号里不写东西的话就不会改变原数组 		还有一个作用:循环遍历，并情况清空数组

空数组.shift（）时，返回undefined，！undefined的Boolean为true，所以undefined本身Boolean为false 那么有了以下格式:

for循环的 for(var arr1 ;arr1 = arr.shift();){ console.log(arr1); }

while循环的 var arr1; while( arr1 = arr.shift()){console.log(arr1)}
