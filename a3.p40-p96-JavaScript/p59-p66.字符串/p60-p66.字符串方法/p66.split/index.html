字符串方法 split

用处: 用于分割字符串，以数组形式返回
格式: str 字符串变量名 str.split("分割符",长度参数)
str.split("")
str.split()
描述:
  • 可以规定（字符串中出现的字符为）要移除的分割符，然后以数组形式返回没有分割符的字符串
  • 长度参数可以省略，长度参数规定的是数组长度n，返回数组长度前n的子字符，大于该长度n不会被返回（也就是大于数组个数n的不要）

    长度参数为0时，返回空数组[]

    长度参数大于最大数组长度时，仍旧按照最大长度来返回，也就是 相当于省略长度参数

  • 分割符为空字符串时，字符串会被全部分割，以数组形式返回所有单个子字符
  • split()空格内为空时，代表没有规定分割符，以数组形式返回完整的字符串
