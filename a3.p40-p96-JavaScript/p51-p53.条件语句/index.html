条件语句

类型 格式 描述 注意事项
if语句 if (返回值为布尔值的式子){语句块} 是则执行,不是则跳过 注意式子里= == ===的区别,=的返回值不是布尔值不要用,可以执行(因为if()会强制转换为布尔值)但代码逻辑错误
if~else语句 if (返回值为布尔值的式子){语句块}else if(...){...}else{语句块}; 是则执行,不是则轮到下个判断,都不是则执行else的语句块 if只与同级最近的else匹配
switch语句 switch(变量){case:值1 语句块;break;case:值n 语句块;break;default:语句块} 是则执行,然后跳出选择,不是则轮到下个判断,都不是则执行defalut的语句块 不写break则不会跳出,会从符合的一项开始继续向下执行
不写default容易漏情况,并不算不合法,default之后的break不写,因为一般default就在最后
