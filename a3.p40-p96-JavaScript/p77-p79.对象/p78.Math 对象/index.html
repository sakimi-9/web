Math 对象

描述 Math对象是js的原生的数学库，提供各种各样的数学功能
可以调用对象中各种各样的函数，调用格式就是调用对象中的函数,括号里输入参数

Math.函数名（参数）
Math对象中常用的方法
  • Math.abs() 取绝对值


  • Math.max() 取其中最大值
    Math.min() 取其中最小值


  • Math.ceil() 向上取整（有小数部分就向上入1）
    Math.floor 向下取整（不管小数多大，都舍去）

    • ceil(天花板) floor(地板)
  • Math.random() 取大于等于0小于1的随机数 可由此取min-max的随机数:
    function 函数名(min,max){ return Math.random()*(max-min)+min; }

console.log(Math.abs(-1)); //取绝对值 为1 console.log(Math.max(1, 2, 3, 66, 777, 888, 34));//取其中最大值 为888 console.log(Math.min(1, 3, 2, -9));//取其中最小值 为-9 console.log(Math.ceil(10.00001));//向上取整 有小数就入1 为 11 console.log(Math.floor(10.999999));//向下取整 不管小数部分多大 都舍去 为10 // 输出 1-100的随机整数 function g(min, max) { return Math.floor(Math.random() * (max - min) + min); } //模拟 100里面抽到99要多少次 while (a = g(1, 100) != 99) { console.log(a); } console.log("99");
