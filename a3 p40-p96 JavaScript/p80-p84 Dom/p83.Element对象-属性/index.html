Element 对象-属性

描述:Element是Dom树上的一个常用大节点。Element对象对应着网页的HTML元素节点的对象。每个HTML元素节点的对象，在Dom树上都会转化为一个Element节点对象

Element 元素节点中常用的对象与属性
属性 id 格式:标签变量名.id
用处:修改该标签id的id名
属性 className 格式:标签变量名.className
用处:修改该标签类class的类名，也可以修改为多个类名（空格隔开）
对象 classList 格式:标签变量名.classList.方法
用处:对该标签class类名进行独立修改

classList有以下方法
  1. add() :增加一个class类名
  2. remove() :删除一个class类名
  3. contains() :检测该元素节点是否含有指定的class类名
    有则返回true 无则返回false
  4. toggle() :检测该元素节点是否含有指定的class类名,有则删除，无则添加 | 相当于把上面三个方法整合到了一起
属性 innerHTML 格式:标签变量名.innerHTML / 标签变量名.innerHTML = "内容"
后面不写内容时，返回该标签的内容 后面用内容赋值时，将修改该标签的内容

可识别内容为标签，所以内容不仅可以是文本字符串 也可以是图片 超链接 等等
内容要为图片 超链接时，用标签的形式写入 比如“《a href=""》《/a》”
属性 innerText 格式:标签变量名.innerText / 标签变量名.innerText = "内容"
后面不写内容时，返回该标签的内容 后面用内容赋值时，将修改该标签的内容

只能识别内容为字符串，所以内容只能是字符串文本
