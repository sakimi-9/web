Dom 概述 (教学视频)

定义 Dom (document object model) 文档对象类型，是Javascript 控制网页的接口。
Dom树与节点 节点 DOM的最小组成单位叫做节点（node)。
文档的树形结构（DOM树）,就是由各种不同类型的节点组成。每个节点可以看作是文档树的一片叶子
    节点的类型有七种
  1. Document:整个文档树的顶层节点
  2. DocumentType:doctype标签
  3. Element:网页的各种HTML标签
  4. Attribute:网页元素的属性（比如class="right")
  5. Text:标签之间或标签包含的文本
  6. Comment:注释
  7. DocumentFragment:文档的片段

Node.nodeType属性 通过以下的返回值，来判断此时操作的是什么类型
