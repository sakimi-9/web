Element属性 获取元素位置

属性 描述
clientHeight 获取元素高度包括 padding 部分，但是不包括 border、margin
clientWidth 获取元素宽度包括 padding 部分，但是不包括 border、 margin
scrollHeight 元素总高度，它包括padding,但是不包括border、 margin包括溢出的不可见内容
scrollWidth 元素总宽度，它包括padding,但是不包括border、margin包括溢出的不可见内容
实际开发环境中，大部分情况，这一对与上面那对没有区别，因为不会设置隐藏内容
scrollLeft 元素的水平滚动条向右滚动的像素数量
scrollTop 元素的垂直滚动条向下滚动的像素数量
offsetHeight 元素的CSS垂直高度（单位像素）,包括元素本身的高度、padding和border
offsetWidth 元素的CSS水平宽度（单位像素）,包括元素本身的高度、padding和border
offsetLeft 到定位父级左边界的间距
没有定位父级则以顶层文档为基准
offsetTop 到定位父级上边界的间距
没有定位父级则以顶层文档为基准
格式:变量名.获取位置的属性

获取视口高度 document.documentElement.clientHeight 视口的高度是当前屏幕窗口的高度

获取页面高度 document.body.clientHeight 页面的高度是内容所撑开的高度


让浏览器自带的外边距（水平的）8px失效，使用以下全局声明
*{padding:0；margin:0；}
