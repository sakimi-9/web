变量

具象化解释:

  • 变量本质:固定空间
  • 变量名:空间名称,用于通过名称操作空间
  • 数据:空间中变化的量

变量提升:js引擎的工作方式是,先解析代码,获取所有被声明的变量(此时还没有赋值的),再一行一行的运行。
(所有变量声明的语句【此时还没有赋值】会被提升到代码的顶部)


以下 情况不会报错,值为undefined(未定义的,未知的)

console.log(n);
var n = 5 ;

实际情况为:
var n;
console.log(n);
n = 5 ;

var 声明变量的关键字

var n = 5; 这是在声明变量

n = 5; 变量重新赋值为

console.log() 输出语句(控制台.打印())

console.log(n) ; 打印n的值
