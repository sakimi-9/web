字符串

  • 组成:是由零个或多个字符组成的，放在单/双引号中
  • 引号嵌套:异种的引号可以直接相互嵌套 | 比如 “ ' ' ” 或 ' “ ” '
    但同种的不能直接嵌套，因为最近的引号会进行配对，导致中间的字符会空出来，从而报错
    要想同种引号嵌套不报错，则需要转义
  • 转义:用于引号嵌套转义或字符串换(分)行转义，在每个引号前或者需要换行处插入反斜杠\
  • 字符串换行:代码中，字符串默认只能写在一行中，分成多行会报错
    要想换行不报错，则需转义
  • length:字符串的length属性，用于读取字符串的长度（字符个数），只读属性
    格式:document.write(字符串的变量名(字符串也行).length)
