字符串方法 slice

slice
用处: 用于返回字符串中位置1至位置2（不包括位置2）的字符，也不会改变原先字符串
根substring的用处一样，但在细节上有所区别
格式: str代表字符串变量名， str.slice(位置1,位置2)
描述:
  • 到位置2但不会返回位置2，起始位置为0的话，终点位置是n就返回前n位
    填起始(小)到结束(大位置)时，和substring的作用一模一样
  • 如果 只写位置1，则会返回位置1至结束所有的字符
    只填起始位置参数时，和substring的作用一模一样
  • 上面两个与substring作用一样的，下面的有区别
  • 如果 位置1比位置2大，则返回空字符串
    substring会进行参数位置交换
  • 如果 位置参数有负数的，则以倒数的为起始位置
    substring会将负数当成0
