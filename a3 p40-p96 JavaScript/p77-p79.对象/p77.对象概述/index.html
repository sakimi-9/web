对象概述

object的定义 一组键值对的集合，一种无序复合数据的集合
描述 键（名）又被称为属性（名）
对象可以储存各种各样的数据，包括函数（在对象中也被称为方法），也包括嵌套对象

存储函数时，函数名写在function（）前
声明对象的语法格式 var 对象名 = { 属性:值， 属性（函数名）:function（）{函数体}， 属性（对象名）:{键值对集合，} }
对象内属性读取方式 只写对象名，键值对会被全部读取

对象名.属性名
对象名.属性名（函数名时）()
记得写括号

调用对象中的函数时，后面不写括号的话，会把:后的全部显示
链式调用 调用对象内嵌套对象的属性，称作链式调用

格式:
对象名.嵌套的对象名.其中的属性名
