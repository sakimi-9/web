Event 事件对象

定义:

事件发生后，会产生一个相应的事件对象，作为参数传给监听函数
描述:

事件的所有信息是以对象的数据类型存储，将事件对象里的所有信息作为参数传给此事件的监听函数，
在事件触发时，监听函数能够调用事件对象的属性及方法，来让事件触发满足更多需求
格式:

Dom0级的 元素标签.事件类型 = function(event){event.属性/方法（）；事件处理}
event可以简写成e , 用event有跳字提示


Event 中常用属性
  • target: 返回此事件作用的标签及其所有信息

  • type: 返回此事件的事件类型



    Event 中常用方法
  • preventDefault(): 阻止默认事件
    解释:事件触发时，阻止该标签的默认效果响应
    比如:click触发时，a标签的跳转效果失效


  • stopPropagation(): 阻止事件冒泡
    解释:子容器的事件触发时，阻止冒泡传递给父容器，让父容器不能同步触发该事件
    比如:子容器的click触发时进行事件处理，父容器同步生效失败
