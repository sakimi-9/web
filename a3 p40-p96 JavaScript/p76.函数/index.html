函数简介

定义 一段可以反复调用的代码块
函数的声明 function 函数名(参数){函数体;return 返回值 }

返回值属于函数体中最后一条语句,返回值这条语句后不要写语句了，因为写了不会被执行

参数不写也可以，代价是函数不好灵活调用了
声明函数时写完函数体后在参数处写好指定参数，方便调用时给指定参数分别赋值
函数的调用 调用格式: 函数名（） ;

说明:函数的声明有两个接口，参数作为入口，传入参数数据，（运行函数体），返回值作为出口，传出运行结果
函数名提升 js引擎视函数名同变量名，运行代码时会将函数声明提升至代码头部，声明中的函数体会被运行

变量名提升，声明会被提升至头部，但此时不会被赋值，而函数名提升，其中的函数体会被运行
