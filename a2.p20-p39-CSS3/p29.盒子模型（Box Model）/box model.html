box model,让页面更加整洁美观

box model 的元素
元素种类 描述 值
content 内容 文本·图片等 (单词不用写)填充的就是内容
padding内边距 设置content与border的距离,padding是透明的,设置大小后会撑开原来的容器
  • padding(一个值时)设置的是内容与边框四个方向的距离
  • padding-(top·bottom·left·right)【常用,清晰明了,方便控制】
  • padding(两个值时):上下 左右
  • padding(三个值时):上 左右 下
  • padding(四个值时 顺时针):上 右 下 左
border边框设置 padding与margin中间夹着的边框 线的粗细npx(空格)线的形式(空格)线的颜色 |CSS里的border有并列的三个值,这三个值都要设置才能使样式生效
线的大小要带单位,线的形式solid(实线)为常用,线的颜色也不要忘
margin外边距 设置该容器与另一个容器的距离,margin也是透明的,设置大小后会撑开容器之间的距离 值的设置方式与padding的一样
