CSS常用新特性 圆角与阴影

圆角 border-radius border-radius:值
值有两种设置方式 npx 和 n%
  1. 一次性设置一个值时 所有角都圆 npx 或 n% | 50%和100% 时为圆
  2. 一次性设置两个值时 左上-右下对角 右上-左下对角
  3. 一次性设置三个值时 左上角 另一对角 右下角
  4. 一次性设置四个值时 (顺时针) 左上 右上 右下 左下
阴影 box-shadow box-shadow:h-shadow(px) v-shadow(px) blur(px) color
box-shadow:水平偏移量(必选) 垂直偏移量(必选) 模糊度(向外模糊的距离,可选) 模糊颜色(可选)
    偏移量设置:
  • 水平和垂直都为0时 默认向四周蔓延一点
  • 值为负时,水平向左偏移 垂直向上偏移
  • 值为正时,水平向右偏移 垂直向下偏移
  • (常用)水平和垂直偏移量都为正 也就是向右下偏移
