CSS的引入方式:

  1. 内联样式(行内样式):
    1. 缺点:缺乏整体性和规划性(不能通过设置一个来让多个应用),维护成本极高
    2. 使用方式:在HTML元素标签里,以style="样式"的形式运用
    3. 比如:

      内联样式使用示例

  2. 内部样式:
      • 优点:单个页面的CSS代码具有整体性和规划性以及方便维护,方便当单个文档需要特殊的样式时使用
      • 缺点:当多个页面时容易混乱,又不好维护了
    2. 使用方式:CSS代码写在head里的style标签中
    3. 比如:内部样式示例
  3. 外部样式:
    • 优点:非常方便,写或改一次CSS代码可以多个页面同时调用生效
    • 使用方式:在外部写好只含有CSS代码的.css文件,在要使用的页面里,head中用link标签来调用
    • 比如:外部样式示例
