数组的扩展方法

Array.from() 将类数组转化为数组
格式: Array.from(伪数组)		 三种伪数组 | 伪数组只能用数组的下标读取方式，和length属性
  1. arguments这个关键字，可以让函数括号内不写参数入口，也可以读到传入参数的伪数组
  2. NodeList元素节点集合，由document.querySelectorAll(“标签”)返回
  3. 类数组的对象对象里键值对的属性名为字符串形式从0开始，里面还是可以有length属性

    比如:var object = { "0":9, "1":"zs", "2":false, length:3 }
Array.of() 将元素集合转化为数组
格式: Array.of(元素集合)		 比如:Array.of(1,2,3,4,5)

Array()这个数组方法，可以创建n个空间的数组
格式:Array（空间个数）
比如:Array(3) //[,,,]
