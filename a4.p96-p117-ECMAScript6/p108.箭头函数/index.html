箭头函数

格式说明
var fn = (参数)=>返回值 		参数为1个时，括号可以省略
var fn = (参数)=>返回值 参数为0个或多个时，括号不能省略
0个时括号是用来占位的
var fn = (参数)=>{代码块} 后面只有一条语句时，相当于return返回值
后面有多条语句则要将代码块写在{}里
var fn = (参数)=>（{返回值为对象}） 当只有一条语句，且返回值为对象 外面要用括号括起来，否则会报错
----- -----
箭头函数最常用的作用 很简洁的写匿名函数
比如:setTimeout(()=>{代码块},delay)
箭头函数里的this 箭头函数根本没有自己的this，而是引用外部的this，所以这个this指向的是外层作用域所定义的
所以setoutTime()里用箭头函数时，直接指向外层作用域，不用在外层先用this赋值变量，再用变量指代
