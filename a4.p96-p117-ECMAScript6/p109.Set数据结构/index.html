Set数据结构

Set（）的描述 Set（）是一种构造函数，构造出特殊的数组(可以使用数组扩展运算符...)，不能出现重复元素的特殊数组，自身并不是数组array
创建一个空的set数据结构 var s = new Set()
给空Set内添加元素 s.add(数值)
s.add(数组)
s.add(字符串)
forEach(传递变量 = >{代码块} ) forEach将数组内的元素循环遍历，并通过传递变量，将遍历的元素作用代码块
格式:数组.forEach(传递变量 = >{代码块} )

可以用forEach给空set塞入指定去重数组
比如:var s = new Set() ; arr.forEach(x = >{s.add(x)} )
new Set()括号里不会强制转换类型，只是去重 比如 new Set(5,"5") //5,"5"
作用 去重 | new Set()后再往里填入参数
  • 给数组去重
    var arr1 = [...new Set（有重复元素的数组arr）]
  • 给字符串去重
    var str1 = [...new Set(字符串)].jion("")
